# Java Book Downloader & Serializer

A Java program that downloads books from public URLs, stores metadata (title, author, URL), and serializes the content locally using object streams.

## Features
- Downloads book content from online sources (e.g., Project Gutenberg)
- Saves metadata and full text in serialized form
- Reads the serialized file back and displays book info
- Demonstrates Java OOP, file I/O, and web content handling

## Technologies
- Java
- Object Serialization
- File I/O
- URL reading

## How to Run
- Compile all `.java` files inside the `src` folder
- Run `Main.java`
